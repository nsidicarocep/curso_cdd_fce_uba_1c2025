🌟 NIVEL 1: PRIMEROS PASOS (15-20 minutos)

Ejercicio 1.1: Tu primer gráfico

Objetivo: Crear un scatter plot básico

Consignas:

  1. Usá el dataset mtcars
  2. Creá un gráfico de puntos con mpg en X y hp en Y
  3. Agregale un título apropiado
  4. Cambiá las etiquetas de los ejes al español

Pistas:

  • Usá ggplot() + aes() + geom_point()
  • Usá labs() para títulos y etiquetas

Ejercicio 1.2: Agregando color

Objetivo: Entender la diferencia entre mapeo y configuración

Consignas:

  1. Tomá el gráfico anterior
  2. Hacé que todos los puntos sean de color azul
  3. Después, cambiá para que el color represente el número de cilindros (cyl)
  4. ¿Cuál es la diferencia entre los dos enfoques?

Ejercicio 1.3: Personalización básica

Objetivo: Aprender parámetros básicos de geom_point

Consignas:

  1. Creá un scatter plot de wt (peso) vs mpg (eficiencia)
  2. Hacé los puntos más grandes (size = 3)
  3. Hacelos semi-transparentes (alpha = 0.7)
  4. Cambiá el tema a theme_minimal()

🚀 NIVEL 2: EXPLORANDO GEOMETRÍAS (20-25 minutos)

Ejercicio 2.1: Histograma informativo

Objetivo: Crear histogramas y agregar líneas de referencia

Consignas:

  1. Creá un histograma de la variable mpg de mtcars
  2. Usá 15 bins
  3. Agregá una línea vertical roja que muestre la media de mpg
  4. Agregá una línea vertical azul que muestre la mediana de mpg
  5. Incluí una leyenda que explique qué representa cada línea

Pistas:

  • Usá geom_histogram() y geom_vline()
  • mean(mtcars$mpg) y median(mtcars$mpg)

Ejercicio 2.2: Boxplots comparativos

Objetivo: Comparar distribuciones entre grupos

Consignas:

  1. Usando mtcars, creá boxplots de mpg para cada número de cilindros
  2. Coloreá cada boxplot según el número de cilindros
  3. Ordená los grupos de menor a mayor número de cilindros
  4. Agregá un título descriptivo

Pistas:

  • Usá factor(cyl) para tratar cilindros como categórica
  • El color va en fill = factor(cyl)

Ejercicio 2.3: Scatter plot con tendencia

Objetivo: Combinar geometrías y entender métodos de suavizado

Consignas:

  1. Creá un scatter plot de wt vs mpg
  2. Agregá una línea de tendencia lineal (method = "lm")
  3. Hacé que la línea sea roja y sin banda de confianza
  4. Coloreá los puntos según el número de cilindros
  5. ¿Qué observás sobre la relación peso-eficiencia?

🎨 NIVEL 3: ESCALAS Y TEMAS (25-30 minutos)

Ejercicio 3.1: Personalizando escalas de color

Objetivo: Controlar paletas de colores manualmente

Consignas:

  1. Creá un scatter plot de hp vs mpg coloreado por cyl
  2. Definí colores manuales: 4 cilindros = verde, 6 = naranja, 8 = rojo
  3. Cambiá el nombre de la leyenda a “Número de Cilindros”
  4. Mové la leyenda abajo del gráfico

Pistas:

  • Usá scale_color_manual()
  • legend.position = "bottom" en theme()

Ejercicio 3.2: Tema personalizado

Objetivo: Crear un tema personalizado

Consignas:

  1. Tomá cualquier gráfico que hayas hecho
  2. Personalizá los siguientes elementos:
    • Título: tamaño 16, negrita, centrado
    • Texto de ejes: tamaño 12
    • Fondo del panel: gris muy claro
    • Líneas de grilla: solo las principales, color gris claro
    • Quitar grilla menor

Ejercicio 3.3: Escalas de ejes

Objetivo: Controlar límites y etiquetas de ejes

Consignas:

  1. Usando datos de wage1, creá un scatter plot de educ vs wage
  2. Limitá el eje Y para mostrar solo salarios entre 0 y 20
  3. Hacé que el eje X muestre solo los valores 8, 12, 16, 18 años de educación
  4. Formateá el eje Y para mostrar valores como dólares (ej: $15)

Pistas:

  • Usá scale_y_continuous() y scale_x_continuous()
  • scales::dollar es útil para formato de moneda

🔥 NIVEL 4: FACETAS Y MÚLTIPLES VARIABLES (30-35 minutos)

Ejercicio 4.1: Explorando con facetas

Objetivo: Usar facetas para comparar subgrupos

Consignas:

  1. Usando mtcars, creá un dataset que incluya una variable de transmisión:
    • am = 0 → “Automática”

  3. Permití que las escalas sean independientes en cada faceta
  4. Coloreá los puntos según el número de cilindros

Ejercicio 4.2: Gráfico multivariado complejo

Objetivo: Mapear múltiples variables a diferentes aesthetics

Consignas:

  1. Creá un scatter plot con las siguientes características:
    • X: wt (peso)
    • Y: hp (potencia)
    • Color: cyl (cilindros)
    • Tamaño: mpg (eficiencia)
  2. Agregá una línea de tendencia
  3. Personalizá la escala de tamaño para que vaya de 2 a 8
  4. Usá la paleta de colores viridis
  5. ¿Qué patrones observás?

Ejercicio 4.3: Análisis de distribuciones

Objetivo: Comparar distribuciones de múltiples maneras

Consignas:

  1. Usando wage1, compará la distribución de salarios entre hombres y mujeres
  2. Creá TRES gráficos diferentes:
    • Histogramas superpuestos (con transparencia)
    • Boxplots lado a lado
    • Gráficos de densidad superpuestos
  3. ¿Cuál comunica mejor las diferencias?
